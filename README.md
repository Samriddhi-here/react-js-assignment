# React JS Assignment

## Project Overview
This is a simple React JS application created as part of an internship assignment.  
The project demonstrates the basic concepts of React such as components, props, state, and event handling.

## Features
- Functional React components
- useState hook for state management
- Passing data using props
- Adding and displaying items dynamically
- Basic conditional rendering

## Tech Stack
- React JS
- JavaScript
- HTML & CSS

## How to Run the Project
1. Clone the repository
2. Open the project folder in VS Code
3. Run the following commands:

```bash
npm install
npm start
